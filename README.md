# Awesome LLM for Scientific Discovery 
 
> 本项目系统汇集了基于大语言模型（LLM）驱动的科学发现相关的研究论文、工具平台和实践案例，涵盖从问题提出、假设生成、实验设计、科学交流到端到端自动科学发现的全过程。旨在为科研工作者提供一个集中、系统、更新及时的资源库。

## 问题提出 (Problem Generation)

  研究问题协同生成的LLM代理框架。

## 领域应用 (Domain Applications) 
 
- **生物与生命科学**：蛋白质结构预测、药物发现等。 
- **化学与材料科学**：催化剂设计、新材料发现。 
- **物理与数学**：公式自动发现、数学推理。 
- **气候与地球科学**：气候预测模型、地震监测。 
- **交叉学科与社会科学**：社科事件分析、金融预测。 
- **计算机科学与人工智能**：AI算法优化、模型设计。 
 
## 平台与工具链 (Platforms ^& Toolchains) 
 
- **AlphaEvolve、ChemCrow、AlphaFold** 等自动科研平台。 
- **SciBench、PaperRobot** 等数据与评估平台。 
 
## 挑战与风险 (Challenges ^& Risks) 
 
- 数据和知识偏差问题。 
- 推理结果的可验证性。 
- 自动化带来的黑箱风险。 
- 人机协同协议与角色设计。 
- 多学科知识交叉融合难题。
